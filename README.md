## 🎯 Project Overview

Financial institutions must assess whether customers can repay loans and credit obligations.

This project predicts customer default probability using machine learning and generates a risk score that assists lending and credit approval decisions.

---

## 💡 Business Problem

Traditional credit assessment methods can be slow and prone to human bias.

This solution provides:

✅ Automated risk assessment

✅ Default probability prediction

✅ Customer risk segmentation

✅ Credit approval recommendations
